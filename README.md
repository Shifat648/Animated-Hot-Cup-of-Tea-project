# Steaming Tea Cup Animation ☕

A pure HTML and CSS art project featuring a 3D-styled tea cup resting on a plate with a realistic, continuous steaming animation. This project demonstrates the power of CSS gradients, border-radius manipulation, and keyframe animations to create engaging visual effects without relying on JavaScript or external graphic assets.

## ✨ Features

* **Pure CSS Animation:** Uses `@keyframes` to create a smooth, infinitely looping vapour effect.
* **Custom CSS Variables:** Utilizes inline CSS variables (`--i`) in the HTML to stagger the animation delays of individual steam particles, creating a natural, randomized look.
* **3D Styling:** Heavily relies on linear and radial gradients, alongside strategic box shadows and pseudo-elements (`::before`, `::after`), to give the flat HTML elements realistic depth, curves, and lighting.

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🚀 How to Run

1. Clone this repository or download the source code files.
2. Ensure both `index.html` and `style.css` are in the same directory.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge). No local server is required!
